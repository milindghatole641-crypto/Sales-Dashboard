This project is an interactive Sales Dashboard built using Microsoft Power BI to analyze sales performance across multiple dimensions such as time, geography, product categories, and shipping modes.
The dashboard helps stakeholders quickly understand business trends, profitability, and top-performing products. Sales-Dashboard
The dashboard provides key business insights including total sales, total profit, number of orders, and product count, along with detailed visual analysis like monthly sales trends, top-performing products, country-wise sales distribution, and category-wise performance.

Advanced features such as DAX calculations, Power Query transformations, slicers, and interactive visuals are used to enable dynamic filtering and better decision-making. This project demonstrates strong skills in data visualization, data modeling, and business analytics using Power BI.
